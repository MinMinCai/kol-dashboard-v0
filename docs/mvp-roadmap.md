# MVP 路線圖（12 週）

## Week 1-2: 基礎建模
- 建立 PostgreSQL schema
- 完成 KOL、社群、合作歷史 CRUD
- 建立權限角色（admin, pm, sales, analyst）

## Week 3-4: 搜尋與篩選
- 建立 KOL 多條件篩選 API
- 建立排序規則（粉絲、互動、歷史成效）
- 建立前端列表與詳情頁

## Week 5-6: 提案流程
- 建立 Proposal 與 Stage 流程
- 記錄客戶偏好與反饋
- 提案版本歷史

## Week 7-8: 委刊單執行
- 建立 insertion order 與任務節點
- 追蹤交付、審稿、付款狀態
- 提醒機制（逾期、待審、待付款）

## Week 9-10: 成效追蹤
- 匯入成效數據（手動/CSV/API）
- 建立 campaign KPI 面板
- 結案評價與復盤欄位

## Week 11-12: AI 功能
- 自然語言搜尋 KOL
- 生成提案摘要與結案報告
- 建立模型評估與人工覆核流程
